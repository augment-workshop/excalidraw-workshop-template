## ðŸ¤– Using Auggie (AI Assistant)

Auggie is your AI coding assistant. Here's how to get started:

### Opening Auggie
- Click the Auggie icon in the sidebar, or
- Use the keyboard shortcut: `Cmd+Shift+A` (Mac) / `Ctrl+Shift+A` (Windows/Linux)

### Example Prompts
- "Explain the architecture of this codebase"
- "Help me understand how the canvas rendering works"
- "Add a new tool to the toolbar"
- "Write tests for the shape selection logic"

### Tips for Effective AI Assistance
1. **Be specific** - Describe exactly what you want to achieve
2. **Provide context** - Reference specific files or functions
3. **Iterate** - Refine your prompts based on responses
4. **Review carefully** - Always review AI-generated code
